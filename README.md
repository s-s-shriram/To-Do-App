# React Native To-Do App

A simple **React Native To-Do App** with:

- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Tasks saved locally using **AsyncStorage**
- Built using **Expo** and React Native

---

## Features

1. **Add Task**: Type in input field and click ADD.  
2. **Edit Task**: Click ✏️ icon → change task → click UPDATE.  
3. **Delete Task**: Click ❌ icon → confirm deletion.  
4. **Complete Task**: Tap task text → toggle completed state (strike-through).  
5. **Persistent Storage**: Tasks saved even after closing the app.

---

## Installation

```bash
git clone https://github.com/YourUsername/react-native-todo.git
cd react-native-todo
npm install
npm start
```
Install Expo Go on your phone to run via QR code, or run in emulator.
